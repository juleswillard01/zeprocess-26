ARCHITECTURE AT A GLANCE
================================================================================

System: 3 Autonomous AI Agents for Sales Automation
Domain: Coaching / Personal Development
Scale: 100-1000 leads/month, 20+ concurrent conversations
Budget: $3,000-$10,000/month (API costs)

Three-Agent Pipeline:
┌─────────────────────────────────────────────────────────────────────┐
│ LEAD ACQUISITION → SEDUCTION (RAG) → CLOSING → PAYMENT              │
├─────────────────────────────────────────────────────────────────────┤
│ • Score ICP match    • Build rapport      • Handle objections        │
│ • Detect spam        • RAG-powered DMs    • Create urgency           │
│ • First contact      • Qualify prospects  • Close deals              │
│ • Handoff ready      • Escalate if ready  • Process payments         │
└─────────────────────────────────────────────────────────────────────┘

Core Technologies:
• Claude Code SDK (Opus/Sonnet/Haiku with intelligent routing)
• LangGraph (multi-agent orchestration)
• LangChain (memory, tools, RAG)
• PostgreSQL 16 + pgvector (database + vectors)
• Python 3.12 + uv (development)
• Docker + Kubernetes (deployment)
• LangFuse + Prometheus + ELK (observability)
• Stripe (payment processing)

KEY ARCHITECTURAL DECISIONS
================================================================================

1. Claude Code SDK as Core Engine
   → Native tools, token counting, production-ready

2. Complexity-Based Model Routing
   → Haiku for filtering, Sonnet for agents, Opus for strategy
   → Result: 60-70% cost savings

3. LangGraph for Multi-Agent Coordination
   → Explicit state management, visual debugging

4. PostgreSQL + pgvector (Unified DB)
   → Cost-effective, transactional + vector storage

5. RAG over Fine-Tuning
   → Immediate deployment, no training cycle

6. V-Code Automated Safety
   → 90% error catch rate, scales automatically

7. SaaS Monthly Licensing
   → $200/month base, enterprise packages

All trade-offs intentionally accepted given constraints.
